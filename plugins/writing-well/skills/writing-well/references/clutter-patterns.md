# Clutter Patterns

Before/after examples for pattern-matching during editing. Every example is drawn from business, email, or web copy — not literary fiction.

---

## Redundant Words

| Before | After |
|--------|-------|
| completely eliminate | eliminate |
| absolutely essential | essential |
| basic fundamentals | fundamentals |
| close proximity | proximity |
| end result | result |
| future plans | plans |
| past history | history |
| added bonus | bonus |
| advance planning | planning |
| free gift | gift |
| new innovation | innovation |
| true fact | fact |
| jointly cooperate | cooperate |

## Prepositional Bloat

| Before | After |
|--------|-------|
| in order to | to |
| in the event that | if |
| at this point in time | now |
| for the purpose of | to / for |
| in the process of | (delete — use the verb) |
| with regard to | about |
| in spite of the fact that | although |
| due to the fact that | because |
| on a daily basis | daily |
| in the near future | soon |
| at the present time | now |
| in a timely manner | promptly / on time |
| with the exception of | except |
| in the amount of | for |
| on the occasion of | when |

## Passive to Active

| Before (passive) | After (active) |
|-------------------|----------------|
| The report was written by the team | The team wrote the report |
| Mistakes were made in the process | We made mistakes |
| The decision was reached by management | Management decided |
| It was determined that costs should be cut | We determined we should cut costs |
| The product was launched successfully | We launched the product successfully |
| Users are encouraged to update | Update your software |
| Payment is required by the 15th | Pay by the 15th |
| Your request has been received | We received your request |

---

## Abstract to Concrete

| Before (abstract) | After (concrete) |
|--------------------|------------------|
| We need to make a decision | We need to decide |
| Please provide an explanation | Please explain |
| We'll conduct an investigation | We'll investigate |
| Give consideration to this option | Consider this option |
| We performed an analysis of the data | We analyzed the data |
| This will have an impact on revenue | This will affect revenue |
| We're in the process of reviewing | We're reviewing |
| He has the ability to lead | He can lead |

## Corporate to Human

| Before (corporate) | After (human) |
|---------------------|---------------|
| We at Acme Corp are pleased to announce | We're launching |
| We are committed to providing excellence | We work hard to be great at this |
| We leverage cutting-edge solutions | We use modern tools |
| Our team is passionate about delivering value | We build things people need |
| Please don't hesitate to reach out | Email us anytime |
| We would like to take this opportunity to thank | Thank you |
| Moving forward, we will endeavor to | We'll |
| Per our previous conversation | As we discussed |
| Please be advised that | (delete — just say it) |
| I wanted to circle back on this | Following up on |

## Throat-Clearing to Direct

| Before | After |
|--------|-------|
| It is interesting to note that sales increased 20% | Sales increased 20% |
| I wanted to reach out because we have a problem | We have a problem |
| I'm writing to let you know that the deadline moved | The deadline moved to March 15 |
| It should be pointed out that this approach has risks | This approach has risks |
| As you may already know, we're launching a new product | We're launching a new product on Tuesday |
| The purpose of this email is to inform you that | (delete — just say it) |
| I just wanted to quickly touch base regarding | (delete — state the topic) |
| Hope this email finds you well. I wanted to discuss | Let's discuss |

## Weak Endings to Strong Stops

| Before | After |
|--------|-------|
| So in conclusion, I think we should go with option A for all the reasons stated above. | Go with option A. |
| I hope this information has been helpful to you and please feel free to reach out if you have any questions. | Questions? Email me. |
| Thank you for taking the time to read this proposal and we look forward to hearing from you soon. | We look forward to your response. |
| That's basically all I wanted to say about this topic. | (delete — you're already done) |
| In summary, the key takeaway here is that we need to act fast. | We need to act fast. |
